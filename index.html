Git Workshop

This is the set of instructions, labs, and follow-along-with-the-instructor script for Matthew McCullough's Git courseware.

This course is taught via GitHub, No Fluff Just Stuff, JAX, SDC, 33rd Degree, DevNexus, JFokus and many other conferences and private events.

Script

"The Git Workshop Student Instructions" file is meant as both a shortcut guide to the common workflows of Git as well as a series of steps you can copy-and-paste from to follow along as Matthew presents the class materials.

Other Git Links

If you want to explore Git to a maximum depth, I recommend Scott Chacon's free Git ebook, ProGit. I also curate a series of Git links via Delicious bookmarks.

Git Training at GitHub

My awesome full time gig at GitHub allows me to offer free Office Hours and Getting Started classes in addition to full-fledged on-site Git Training events.
